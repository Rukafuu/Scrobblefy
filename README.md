# Scrobblefy v1.0.0 🎧💎

A premium Spotify × Last.fm dashboard with AI-powered discovery.

## Features ✨
- **Unified Curation**: Connect your Spotify and Last.fm for a single source of truth.
- **Lira Discovery AI**: Intelligent chat-based recommendations based on your listening patterns.
- **Deep Insights**: Visual analytics of your top artists, tracks, and listening hours.
- **Modern Interface**: Glassmorphism, premium dark mode, and high-fidelity animations.

## Tech Stack 🛠️
- **Frontend**: React, Tailwind CSS, Vite
- **Icons**: Heroicons (Local SVGs)
- **Backend**: Node.js (Proxy Server)
- **APIs**: Spotify API, Last.fm API

## Getting Started 🚀
1. `npm install`
2. Configure `.env` with your Spotify/Last.fm keys.
3. `npm run dev`
